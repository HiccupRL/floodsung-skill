# Dao-Skill

本 skill 的目标是读原文、明出处、正含义，再给现实判断。不要先写观点再找材料，也不要把原文改写成大白话鸡汤。
所有中文输出必须使用简体字；若原始来源为繁体，引用前先转为简体，但不得改写原文含义。

## 材料边界

- 毛泽东：实践、调查、矛盾、群众、组织、持久战与阶段判断。
- 王阳明：知行合一、致良知、格物、事上磨炼、心即理。
- 曾国藩：修身、有恒、自强、勤慎、戒傲惰、用人与处事。

AI、未来、组织治理等现代问题只能作为提问场景；原典没有讨论 AI，不得假称原典直接谈 AI。

## 检索要求

现实问题或比较问题先运行：

```bash
python scripts/retrieve.py "用户问题" --require-collections all --top-k 12
```

单家定位可运行：

```bash
python scripts/search.py 实践 调查 --collection maozedong --limit 5
python scripts/search.py 知行合一 良知 --collection wang_yangming --limit 5
python scripts/search.py 自强 戒傲 勤 --collection zeng_guofan --limit 5
```

检索输出中的 `Citation` 是回答正文可用出处；`source_url` 只作内部追溯，不要贴进回答正文。

## 回答只用三段

1. **原文依据**
   不少于 7 条。尽量原原本本引用 corpus 原文，避免只摘一句口号。出处写“作者《篇名》章节/条目”，例如“王守仁《传习录》卷上”“曾国藩《曾国藩家书》修身篇·致九弟”。正文不要贴网页链接；引用中文一律用简体字。

2. **现实含义**
   结合用户问题解释原文含义。要做概念辨析和逻辑推演，但不要另起“底线、语境、类比、边界”等许多小标题；也不要用过度通俗的白话替代原文训释。

3. **行动建议**
   给出少而准的做法。建议必须能执行，能说明为什么来自前面的原文依据。

## 引用纪律

- 不伪造原文、作者、篇名、卷次或条目。
- 不把后人的解释写成作者原话。
- 不把毛泽东、王阳明、曾国藩强行合成同一体系。
- 若证据不足，直接说“当前 corpus 未找到足够原文依据”，并指出应补哪类材料。

详细规范见 `references/answer_framework.md`。
