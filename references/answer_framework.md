# 回答框架

目标：让输出同时有原典依据、逻辑深度和现实指导意义。

## 1. 变量定义

先把问题拆成可判断结构：

- 对象：个人、团队、组织、制度、知识生产过程。
- 变量：速度、正确性、责任、风险、权力、学习能力、长期韧性。
- 优化目标：效率、可控性、公共利益、修身成德、组织胜任力或长期生存。
- 约束：证据质量、时间尺度、责任归属、反馈是否可观测。

## 2. 原文依据

默认三家各至少一条。每条应包含：

- 作者
- 标题或篇名
- 来源 URL
- 不超过必要长度的短摘录

原文只说明“它在原语境中说了什么”，不要直接跳到现代结论。

## 3. 语境解释

解释原文所处理的问题：

- 毛泽东：实践、调查、矛盾、群众、阶段、组织。
- 王阳明：心、理、知行、良知、去私欲、事上工夫。
- 曾国藩：修身、持恒、勤慎、戒傲惰、用人、家国事务。

## 4. 类比推论

把现代问题迁移到原典时，只迁移结构，不迁移历史身份：

- 可迁移：实践反馈、知行闭环、权力约束、风险分阶段、长期积累。
- 不可迁移：具体政治立场、战争情境、宗法家庭结构、历史制度条件。

## 5. 逻辑审查

每个结论必须能回答：

- 因是什么，果是什么，中间机制是什么？
- 哪个观察会推翻这个判断？
- 反例是什么？
- 与基线相比，建议优化了什么？

## 6. 行动判断

建议要能落地：

- 给个人：下一步练习、学习、验证或自我约束。
- 给团队：决策流程、责任边界、复盘机制、试点规模。
- 给组织：治理结构、反馈渠道、风险分层、授权与问责。

避免只给“保持清醒”“加强学习”这类不可检验建议。
